# Employee Task Dashboard

A modern Employee Task Dashboard built using React and Vite. This application allows users to manage daily tasks with an easy-to-use interface.

## Live Demo

https://employee-task-dashboard-ten.vercel.app

## GitHub Repository

https://github.com/janvighogale90-cmd/employee-task-dashboard

## Features

- Dashboard with task summary
- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Search tasks by title
- Filter tasks by status
- Responsive design
- Data stored using Local Storage

## Tech Stack

- React.js
- React Router DOM
- JavaScript (ES6)
- HTML5
- CSS3
- Vite
- Local Storage

## Installation

```bash
git clone https://github.com/janvighogale90-cmd/employee-task-dashboard.git

cd employee-task-dashboard

npm install

npm run dev
```
